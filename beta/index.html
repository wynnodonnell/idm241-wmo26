IDM241: Beta
Website being Redesigned: My Website wynnodonnell.com: Mobile menu.
For the beta for this project I want to redo the microinteraction I currently have on my mobile version of my portfolio website. It is a hamburger menu, and I want to change a few things about it.
  • the menu icon animation for when it opens and closes
  • Opening and closing animations for the menu itself
  • Hoverstates inside of the menu
I plan on doing this using a combination of CSS, HTML, and Javascript.

Trigger:

The trigger for this microinteraction takes place when the user clicks on the menu icon.

Rules:

When the user clicks on the menu icon the menu icon will change to something else, the menu will open, and the menu will stay open until the icon is clicked again.

Feedback:

The user will know the microinteraction is happening when they see the menu open and the icon change when they click on the icon.

Modes and Loops:

There are two modes in this microinteraction, open mode and closed mode. In open mode there will be hoverstates over the clickable links in the menu, and the menu will be open and the icon will be different. In the close mode there will be a menu icon visible, but no visible menu.
